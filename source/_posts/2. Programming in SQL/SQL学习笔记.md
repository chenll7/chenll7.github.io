# SQL学习笔记

## 表连接

- 内连接

  - 关键字：`,`、`INNER JOIN`、`JOIN`

- 自然连接
  - 相当于内连接并隐含“WHERE A.field = B.field”
  - 关键字：`NATRUAL JOIN`
- 左连接

  关键字：`LEFT JOIN`

- 右连接

  关键字：`RIGHT JOIN`
