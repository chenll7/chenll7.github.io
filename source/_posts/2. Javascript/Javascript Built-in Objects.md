# Javascript Built-in Objects

## window

　　`window`是Node.js中的全局对象，全局作用域下`this`即指向这个对象。

## console

```js
// Verbose Level
console.debug('123');
// Info Level
console.log('123');
console.info('123');
// Warning Level
console.warn('123');
// Error Level
console.error('123');
```
