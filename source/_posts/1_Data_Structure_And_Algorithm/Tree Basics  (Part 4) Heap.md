# Tree Basics (Part 4): Heap

## Basics

　　时间复杂度为O(logN)。

### Java

```java
// Minimum heap.
Queue<Integer> minHeap = new PriorityQueue<>();
// Maximum heap.
Queue<Integer> maxHeap = new PriorityQueue<>((o1, o2) -> o2 - o1);
// Some operations.
maxHeap.add(132423);
var size = maxHeap.size();
var maxVar = maxHeap.peek();
maxVar = maxHeap.remove();
```
