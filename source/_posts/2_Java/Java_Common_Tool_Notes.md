# Java 常用工具笔记

## 字符串、数组以及数值操作

　　Apache Commons Lang 3。提供`StringUtils`，用于判断是否为空白字符，空字符，小写、大写或字母字符等。提供`ArrayUtils`，用返回给定数组的字符串表示等。

## POJO转化
