# rdesktop 笔记

　　rdesktop是Linux下RDP远程桌面工具，使用例子如下：

```shell
rdesktop -f -a 16 -r sound -r clipboardPRIMARYCLIPBOARD -r diskbear=homebear -r diskdevice=mediabear -x l 10.21.0.251
```
