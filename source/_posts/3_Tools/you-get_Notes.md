# you-get笔记

## 安装

```shell
pip install you-get
```

## 使用

```shell
# 使用HTTP代理下载（不需要加HTTP_PROXY环境变量或者HTTPS_PROXY环境变量）
you-get -x "localhost:1080" "https://www.youtube.com/watch?v=54eCvQLEZVg"
```
