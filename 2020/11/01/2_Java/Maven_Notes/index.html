Maven笔记

概念

  • Lifecycle 内置的有default、clean和site三种Lifecycle。

  • Phase：每个Lifecycle有若干个Phase。

  • Goal：每个Phase对于Plugin来说是一个Goal。

  • Plugin：一个Plugin可以对应一个Goal，或者说一个Phase。

仓库

分为本地仓库和远程仓库。远程仓库分为中央仓库、私服和其他公共库。

设置文件
