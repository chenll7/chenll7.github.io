这个例子成功的将观众提取成了切面。@Aspect注解的好处是可以将一个普通的POJO变成一个切面，例如观众类。观众可以在此类中管理自己的行为。我们只需要监听演出方法就好了，演出之前之后如何调用就交给Spring，很方便。
